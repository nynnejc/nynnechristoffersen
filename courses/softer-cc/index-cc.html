Softer Creative Coding Spring 2025

This website contains the materials for a series of workshops held by Nynne in the spring of 2025 with SOFTER in Copenhagen. These events introduces creative coding with the visual coding framework P5.js (using the JavaScript programming language). You will get to try your hand at using the basic building blocks of programming to draw and animate with code! The event is suitable for absolute beginners, but we can also try to accomodate if you come with more experience and are looking for inspiration for the next steps in your learning.

Each page on this little website has a p5.js sketch imbedded as a background. You can find all the sketches here. All the code was hand-written by Nynne and can also be seen here.
