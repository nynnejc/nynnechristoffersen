setup()

p5.js has its own syntax and built-in methods that we will usually always apply to our drawing. We know that something is a method whern we see it ending with()

One of these is called setup()

Here we put any code that is involved with setting up the initial state of your program: size of you graphics, canvas, the weight of your stroke or the speed of your program.

Example:

function setup() {

    createCanvas(400, 200); // Setting up size of our drawing

}

draw()

draw() is where all the code goes that is involved with drawing to the screen: setting background color, drawing shapes etc.

Example:

function draw() {

    background(220); // setting a background color (gray)

}

Our first sketch

Let's recap what we need in order to make our first drawing with p5.js:

setup() Is called as the first thing when the program runs. Usually contains start variables and functions e.g. Background color and screen size. Can only be called once.

createCanvas() Called only once. Must be placed in the setup() function. Creates a canvas with dimensions in pixels. Without parameters, canvas becomes 100x100 pixels.

draw() Runs the code inserted in the body in a loop until noLoop() is called.

background() Sets the background color. If not used, the background is transparent. A number is understood as grayscale and string as CSS color. Three parameters RGB (default), HSB or HSL.

Basic Shapes

line()

  • Syntax: line(x1, y1, x2, y2)
  • Draws a line
  • Has no weight / size and therefore cannot use fill(). Instead, use stroke() to change the color and strokeWeight() to change the width of the line.

circle()

  • Syntax: circle(x, y, d)
  • Draws a circle
  • The first two parameters define the location of the center of the circle. Third parameter defines the diameter of the circle.
  • Alternative to ellipse()

square()

  • Syntax: square(x, y, d)
  • Draws a square
  • The first two parameters define the location of the square's upper left corner. Third parameter defines the length of the side of the square.
  • Alternative to rect()
